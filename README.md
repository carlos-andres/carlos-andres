# Calo

### Team Lead &amp; Senior Software Engineer — Backend &amp; Integrations

Medellín, Colombia · Remote &nbsp;·&nbsp; [carlos-andres.pro](https://carlos-andres.pro)

I build the backend systems and data pipelines that move inventory between manufacturers,
dealers, and online marketplaces — and I prefer owning whole systems over isolated tasks. Two
decades building for the web, 15+ of them in modern software engineering: Laravel APIs,
distributed PHP workers, Elasticsearch, and infrastructure as code. On my own time I ship small,
sharp developer tools — a native macOS app, browser extensions, terminal and editor tooling —
built increasingly with AI-assisted workflows. Clean architecture and good tooling are the
throughline of everything I make.

## Selected work

Things I've shipped on my own time — small, self-contained tools I actually use.

- **[DotEdit](https://github.com/carlos-andres/DotEdit)** — native macOS app (SwiftUI) for comparing and syncing `.env` files. &nbsp;`Swift`
- **[devbar.wezterm](https://github.com/carlos-andres/devbar.wezterm)** — smart WezTerm status bar: git, runtimes, SSH awareness, direnv, clean UI. &nbsp;`Lua`
- **[workflow-system](https://github.com/carlos-andres/workflow-system)** — a structured development workflow for AI-assisted coding with Claude Code. &nbsp;`Shell`
- **[CSS Helper](https://github.com/carlos-andres/css-helper-site)** — Chrome/Firefox extension that shows the author CSS that actually *won* the cascade for any element, with source filenames. &nbsp;`HTML`

- **[intellij-open-iterated-files](https://github.com/carlos-andres/intellij-open-iterated-files)** — IntelliJ plugin that opens all new/modified files in a Git project. &nbsp;`Kotlin`
